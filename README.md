# python-programming-exercises
Collection of weekly programming exercises and laboratory assignments in Python at Universidade de Aveiro.

## Overview
This repository contains a structured collection of practical exercises and laboratory assignments developed during my coursework at the **Universidade de Aveiro**. 

The main objective of these exercises was to master Python's syntax, core data structures, script automation, and foundational programming logic.

## Repository Structure
The project is organized chronologically by lecture/lab session (*Aulas*):
* `aula1/` to `aula9/`: Weekly script assignments, fundamental algorithms, and problem-solving exercises.

## Concepts Covered
* Control flow and loops
* Data structures (Lists, Dictionaries, Tuples, Sets)
* File I/O operations and scripting
* Code optimization and modular functions
